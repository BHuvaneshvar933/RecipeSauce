# RecipeSauce

## 🍽️ About RecipeSauce
RecipeSauce is a web application designed to help users discover, share, and organize recipes effortlessly. Whether you're a home cook or a professional chef, RecipeSauce provides an intuitive interface to manage your favorite recipes in one place.

## 🚀 Features
- 🌍 Browse a variety of recipes from different cuisines.
- 📜 View detailed ingredient lists and step-by-step cooking instructions.
- 🔍 Search for recipes based on ingredients or dish names.
- ❤️ Save and manage your favorite recipes.
- 📝 Add, edit, and delete your own recipes.

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS, JavaScript, React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **UI Framework:** Tailwind CSS

## 📦 Installation
### 1️⃣ Clone the repository
```bash
 git clone https://github.com/BHuvaneshvar933/RecipeSauce.git
```
### 2️⃣ Navigate to the project directory
```bash
cd RecipeSauce
```
### 3️⃣ Install dependencies
```bash
npm install
```
### 4️⃣ Start the development server
```bash
npm start
```

## 💡 Usage
1. Open your browser and navigate to `http://localhost:3000`.
2. Browse through the recipes or use the search bar to find specific dishes.
3. Click on a recipe to view detailed ingredients and instructions.
4. Create an account to save, add, and manage your favorite recipes.
